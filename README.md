# MY PERSONAL NODEBOILER FOR TYPESCRIPT PROJECT

## **Installation**

Prefer with yarn :

```bash
yarn
```

## How To Running This

**Development**

```bash
yarn start:dev
```

**Production**

```bash
yarn start:build
```

## Installed Package

**Authentication**

1. Jsonwebtoken

2. Passport
   
   1. Local / Username and Password
   
   2. Google / Login with Google

**Database**

1. MongoDB with Mongoose as ODM
